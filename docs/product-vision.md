# Product Vision

## What This Is

Codex Team Bootstrapper is a Codex-first installer for a practical AI team. It generates a file-based operating layer inside a target repository so teams can route work through clear roles, prompts, docs, workflows, and tracked install metadata instead of ad hoc prompting.

## Who It Is For

- engineers shipping modern web products
- teams working in new or existing repositories
- teams that want specialized agents without maintaining a large prompt zoo

## Problem It Solves

Copying prompt files by hand is inconsistent, hard to maintain, and usually disconnected from real repo workflows. Existing repositories also lack a standard way to move work between product, engineering, support, sales, and release activities.

## Product Position

This project is a generator and bootstrap system, not an app repo and not a direct clone of an agent library. Its job is to install a compact, opinionated, and auditable team package that works well with Codex on first use.

## Success Criteria

- a repository can be bootstrapped in minutes
- the generated output is coherent on first install
- the installed team supports feature work, bugfixes, handoffs, and release checks
- brownfield installs stay safe by default
