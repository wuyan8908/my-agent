# Fast Mode Prompt

Use this for small, bounded tasks in `{{projectName}}` under the `{{blueprint}}` blueprint.

## Task

State the task clearly and keep the scope narrow.

## Constraints

- stay within one small change
- do not spawn subagents unless the task clearly grows
- preserve existing repo conventions
- keep the output brief and actionable

## Expected Output

- files changed
- what was done
- any follow-up risk
