# Sales Proposal Request

## Goal

Create a sales proposal that is accurate, concise, and aligned with actual product capability.

## Inputs

- customer profile
- pain points
- proposed solution
- scope boundaries
- pricing assumptions
- timeline expectations

## Expected Output

- executive summary
- problem framing
- proposed solution
- scope and exclusions
- timeline
- risks and assumptions

## Coordination

- `account-strategist` validates customer context
- `sales-proposal-strategist` writes the proposal
- `product-manager` checks for promise drift
