# SEO Optimization Request

## Goal

Improve discoverability without harming clarity or product accuracy.

## Inputs

- page or content target
- primary keyword
- secondary keywords
- audience
- conversion goal
- brand voice constraints

## Expected Output

- title and meta suggestions
- content structure recommendations
- internal linking ideas
- priority fixes
- risks if changes affect UX or copy quality

## Coordination

- `seo-specialist` leads
- `content-creator` rewrites copy
- `product-manager` validates claims
